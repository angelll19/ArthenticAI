# 🎨 Arthentic AI

### AI-Powered Art Discovery & Visualization

**Live Demo:** https://arthenticai.vercel.app/

---

## Overview

Arthentic AI is an AI-powered art discovery platform that helps users explore, understand, and visualize artwork within their own spaces.

The platform combines AI-generated art insights with augmented reality (AR) visualization, allowing users to discover artworks that match their preferences and preview how selected pieces would appear in a real-world environment before making a decision.

By analyzing artistic styles, visual characteristics, and user preferences, Arthentic AI delivers a more personalized and interactive approach to art exploration.

---

## Features

### 🎨 Artwork Discovery

Explore and discover artworks based on your interests and aesthetic preferences.

### 🧠 AI-Powered Art Insights

Gain deeper understanding of artworks through AI-generated analysis of artistic styles and visual elements.

### 📱 AR Visualization

Preview artworks directly within your space using augmented reality to see how they fit your environment.

---

## Vision

Arthentic AI aims to make art discovery more accessible by combining artificial intelligence and immersive visualization, helping users find artwork that aligns with their personal style and living spaces.

## Technologies Used
-Google AI Studio
-Google Gemini API
-Vercel (Deployment)

## Development
This project was built using Google AI Studio and deployed on Vercel. The application leverages Gemini models for AI-powered artwork analysis and recommendations.

---

## Run Locally

### Prerequisites

* Node.js

### Installation

1. Install dependencies:

```bash
npm install
```

2. Set the `GEMINI_API_KEY` in `.env.local`:

```env
GEMINI_API_KEY=your_api_key_here
```

3. Run the application:

```bash
npm run dev
```

4. Open your browser and visit:

```text
http://localhost:3000
```
